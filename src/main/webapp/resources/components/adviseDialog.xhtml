Gratulacje! Założyłeś nowy wniosek.

System ocenił twoją zdolność kredytową na wysoką. Bez problemu powinieneś otrzymać kredyt. System ocenił twoją zdolność kredytową jako średnią.

W celu poprawienia swojej zdolności kredytowej:

Dodaj swój BIK - dzięki analizie raportu BIK system może dokładniej przeanalizować twoją sytuację.
Aktualizuj historię konta - bardziej szczegółowa historia konta bankowego pozwoli dokładniej określić twoje możliwości kredytowe.
Spłać stare debety - nawet niewielkie kwoty pozostałych debetów znacząco obnizają Twoją zdolność.
Znajdź współkredytobiorcę - jeżeli możesz weź kredyt wraz z współkredytobiorcą.
Dodaj inne źródła dochodu - jeżeli posiadasz inny stały dochód również uwzględnij go w przychodach.
Rozważ pożyczkę poza bankami - instytucje parabankowe mają zazwyczaj większe możliwości dotyczące zdolności. System ocenił twoją zdolność kredytową jako niedużą.
